# 🚀 Como Configurar a IA (Groq - GRÁTIS)

## Passo 1: Criar conta no Groq

1. Acesse: https://console.groq.com/
2. Clique em "Sign Up" (pode usar Google/GitHub)
3. É 100% gratuito!

## Passo 2: Gerar sua API Key

1. Após login, vá em: https://console.groq.com/keys
2. Clique em "Create API Key"
3. Dê um nome (ex: "dota-overlay")
4. Copie a chave que aparece (começa com `gsk_...`)

## Passo 3: Adicionar no projeto

1. Abra o arquivo: `dota-overlay\.env`
2. Substitua `seu_key_aqui_quando_criar` pela sua chave:
   ```
   GROQ_API_KEY=gsk_SuaChaveAqui...
   ```
3. Salve o arquivo

## ✅ Pronto!

Agora reinicie o overlay e a IA vai funcionar!

---

## 🎯 O que a IA faz:

✅ Analisa composição de draft automaticamente
✅ Sugere estratégias para cada time
✅ Identifica sinergias e counters
✅ Gera conceitos estratégicos em tempo real

## 📊 Limites Grátis do Groq:

- 6000 requests por dia
- Velocidade: Ultra rápida (melhor que GPT-4)
- Modelo: Llama 3.1 70B (muito bom!)

---

## 🆘 Sem API Key?

Não tem problema! O sistema funciona com análises básicas sem IA.
Quando você configurar, as análises ficam muito melhores!
